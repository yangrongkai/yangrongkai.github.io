#### 商业计划书
1. 你的好的idea是什么？
2. 你为什么说这个idea是具有商业价值的？（趋势）
3. 你需要一个什么样的计划，阶段来支撑、实现这个商业价值？
4. 你当下阶段缺什么？我能得到什么？

```bash
时刻坚定创业的方向及决心
时刻清楚方向的价值
时刻存有应对并解决问题的策略
时刻知悉创业在不同阶段的不足及优势。
```
